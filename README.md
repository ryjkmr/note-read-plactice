# 譜読み練習

ブラウザ上で動作する、MIDI キーボード対応の譜読み練習アプリです。  
出題範囲、調号、譜表、加線ルール、臨時記号表記などを設定し、表示された音符に対応する鍵盤を弾いて練習できます。

## 特徴

- MIDI キーボード入力で解答
- ト音譜表、ヘ音譜表、大譜表に対応
- 調号に応じた音名表記
- 大譜表での中央付近の音の出し分け
- 加線付き出題の ON/OFF と上限本数設定
- 問題譜面の表示倍率切り替え
- 設定内容の保存

## 使い方

1. `index.html` をブラウザで開きます。
2. 出題範囲、調号、譜表などを設定します。
3. MIDI キーボードを接続します。
4. 表示された音符に対応する鍵盤を弾きます。
5. 正解すると次の問題が表示されます。

## 動作環境

- Web MIDI に対応したブラウザ
- MIDI キーボード

`navigator.requestMIDIAccess()` を使用しているため、ブラウザによっては動作しません。


## 依存ライブラリ

- [OpenSheetMusicDisplay](https://opensheetmusicdisplay.github.io/)

現在は CDN 経由で読み込んでいます。

## 保存される設定

以下の設定はブラウザの `localStorage` に保存されます。

- 最高音
- 最低音
- 調号
- 譜表
- スケール外の音を含む
- 近い音の連続を減らす
- 大譜表で加線を使用
- 加線数の上限
- 臨時記号表記
- 問題譜面の表示倍率
